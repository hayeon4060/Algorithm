## BFS는 최단 거리 찾을 때 사용
## DFS는 모든 경우의 수를 구할 때 사용

## 예외 케이스

1. 연결된 노드가 하나도 없을때
2. sort() 해야하며, .sort((a,b)=>{return a-b)} 처럼 숫자여도 정확히
3. 같은 연결이 순서 바뀌어서 두번 나올때 1->2, 2->1 양방향인 경우 (같은 상황 두번)


## BFS
1. queue 사용하여 현재 노드 queue 추가
   
2. queue 첫번째 빼고, result에 push
    shift는 O(n)이기 때문에 queue_point를 사용하면 좋음
   
3. result 에 push된 노드와 연결된 모든 노드를 queue에 차례대로 추가
 
4. 2~3 반복

이중배열 쓰는게 더 좋음. nds[3]: O(1), nds['3']은 처음부터 찾아야 해서 O(n)일 수도
